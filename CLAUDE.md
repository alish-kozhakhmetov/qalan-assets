## Что собираем
Продуктовый лендинг Qalan — AI-тренажёр по математике, 1–11 класс. Аудитория — родитель (платит и читает), рассказываем о продукте в целом. Язык контента — русский, с переключалкой языков. Одностраничник, анимированный.

## Стек и сетап
```bash
npm create vite@latest qalan-landing -- --template react-ts
cd qalan-landing && npm i
npm i framer-motion
npm i -D tailwindcss postcss autoprefixer && npx tailwindcss init -p
```
- **Framer Motion** — слой анимаций.
- **Magic MCP (21st.dev)** — `npx @21st-dev/magic@latest`, требует API-ключ 21st.dev. Подключить как MCP-сервер к Claude Code. Используем, чтобы вытягивать болванки секций, ПОТОМ переодеваем в нашу DS (см. ниже).
- **Скилл `nextlevelbuilder/ui-ux-pro-max-skill`** — сторонний community-плагин. ⚠️ Просмотреть `cli/` и `workflows/` перед установкой, понять что делает. Не блокер, но вшивать чужой плагин вслепую не надо.

## Позиционирование и голос
- **Ядро продукта — персональный AI-тренажёр** (индивидуальный подход, задачи под уровень, разбор ошибок). 90% платных живут в нём.
- **Кешбэк НЕ киллер-фича.** Не вести заголовок деньгами, не обещать вывод. Награды зарабатываются и тратятся внутри игры (персонажи, способности).
- **Self-serve.** CTA «Начать бесплатно» → триал / браузер / сторы. Никаких лид-форм и «оставьте номер».
- **Голос:** тёплый, к родителю на «вы», двойная аудитория («ребёнку — самостоятельно, родителю — следить за прогрессом»), регистр Skysmart, конкретика вместо прилагательных, без корпоративщины.
- **Языки:** контент RU + дропдаун Қазақша / Русский / Oʻzbekcha / Azərbaycanca. KZ/UZ/AZ НЕ переводить самому.

## Анимации (Framer Motion)
- **Hero:** `staggerChildren` — построчный/послов­ный вход заголовка; CTA и устройство — мягкий fade-up; устройство циклит вход→решение.
- **Бенто:** `whileInView` fade + slide-up со stagger по ячейкам; hover — лёгкий lift/shadow.
- **Геймификация:** карточки персонажей — hover scale + свечение; выбранная карточка акцентнее.
- **Тарифы:** hover scale + glow-border на Premium.
- **Глобально:** пружинные easing, сдержанно (не перебор). Уважать `prefers-reduced-motion`.
